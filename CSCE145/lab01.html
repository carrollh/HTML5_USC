  14. A new class file will be created with the basic code you will need for your assignment. Include the following statement at the beginning of the class so that you can use the Scanner class:
    import java.util.Scanner;
    (Make sure you include the semicolon at the end of the line.)
  15. The cursor should be located underneath the line 
public static void main(Strings[] args) {

    System.out.println("Hello out there.");
    //two forward slashes means the rest of the line is a comment!!!!

    //Make a String variable for storing a first name -- change this to your first name
    String firstName = "James";
    //Make a String variable for storing a last name -- change this to your last name
    String lastName = "Bond";

    //Make a char variable that indicates the grade you want to make in this class
    char grade = 'A'; //note single quotes ' around characters

    //Finally, output all of that information to the screen
    System.out.println("My first name is " + firstName + " and my last name is " + lastName +
            " and if I study hard, I will make a(n) " + grade + " in CSCE145!");


    //Prompt the user to input two numbers to add
    System.out.println("I will now add two integer values, please enter the two integer values separated by a space: ");

    //Now create and connect a Scanner object to the keyboard
    Scanner keyboard = new Scanner(System.in);

    //Now use the keyboard object to retrieve values from the user
    int value1 = keyboard.nextInt();
    int value2 = keyboard.nextInt();
    int result = value1 + value2; //This adds the values

    System.out.println("The result of adding " + value1 + " and " + value2 + " is: " + result);

    System.out.println("Goodbye");

If you make a typing mistake you can correct it with the backspace or delete key. If you have positioned the cursor somewhere else, then you can use the four arrow keys to get back to the correct line.

  1. Use the File/Save to save and compile the program. If there is an error while compiling your code it will show up in the tasks window of eclipse workbench with a red cross in the first column, a description, and the line number where it occurs.

  3. Now to compute something! Write a program that computes the volume of an object. The program should ask the user to input the object mass and the density (use integers for this). The mass will be given in grams; the density will be in grams per cubic centimeter. Your program should compute and print on the screen the volume in cubic centimeters. (Note that this is very similar to the program above that you just modified and ran.)  The relationship of mass, density, and volume of an object is given by

Volume = Mass / Density
